# Financial Crime Analytics: AML Transaction Monitoring
## Project Overview
This project is the Capstone project for my Google Data Analytics Certificate course. It demonstrates my ability to apply the full data analysis lifecycle to address complex operational risks in fintech and financial institutions.
* **Objective:** Build transaction monitoring solutions to detect risk indicators.
* **Core Skills:** Cloud Data Ingestion, SQL Data Cleaning, Anomaly Detection, BI Dashboarding.
* **Tech Stack:** Google BigQuery (SQL), Microsoft Excel, Tableau.
* **Data Source:** [Anti Money Laundering Transaction Data](https://www.kaggle.com/datasets/berkanoztas/synthetic-transaction-monitoring-dataset-aml?resource=download)
